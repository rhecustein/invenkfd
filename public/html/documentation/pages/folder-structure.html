Folder Structure

You can see the following folder structure once you have entered starter directory

    • .babelrc
    • gulpfile.babel
    • package.json
    • app
      • assets
        • css
        • es6
        • fonts
        • images
        • js
        • scss
        • vendors
    • v-... .html
    • h-... .html
.babelrc

Babel config file

gulpfile.babel

Included all gulp tasks logic

package.json

npm entry file

app/assets/css

All vallina css files placed here

app/assets/es6

Included all ES6 files which will transpile js folder

app/assets/fonts

Included all font files

app/assets/images

Included all image files

app/assets/js

Included all transpiled JS files

app/assets/scss

Included scss files which will output to css folder

app/assets/vendors

Included all source files of 3rd party plugins

app/v-__.html

html files with vertical layout

app/h-__.html

html files with horizontal layout
